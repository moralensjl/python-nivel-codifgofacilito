## üß© ¬øQu√© es un diccionario?

Un **diccionario (`dict`)** es una estructura que almacena **pares clave-valor**.
Cada **clave** debe ser √∫nica, y sirve para acceder a su **valor**.

```python
persona = {
    "nombre": "Moralens",
    "edad": 26,
    "pais": "Rep√∫blica Dominicana"
}
```

Aqu√≠:

* `"nombre"`, `"edad"` y `"pais"` son las **claves**.
* `"Moralens"`, `26` y `"Rep√∫blica Dominicana"` son los **valores**.

---

## üß† C√≥mo acceder a los valores

```python
print(persona["nombre"])  # Moralens
```

Si la clave **no existe**, genera un `KeyError`.
Por eso se suele usar `get()`:

```python
print(persona.get("profesion", "No especificada"))  # No especificada
```

## üõ†Ô∏è Funciones y m√©todos comunes de diccionarios

| M√©todo / Funci√≥n                    | Descripci√≥n                                     | Ejemplo                                                    |
| ----------------------------------- | ----------------------------------------------- | ---------------------------------------------------------- |
| `len(dic)`                          | Devuelve cu√°ntos pares clave-valor hay.         | `len(persona)` ‚Üí `3`                                       |
| `dic.keys()`                        | Devuelve una vista con las claves.              | `persona.keys()` ‚Üí `dict_keys(['nombre', 'edad', 'pais'])` |
| `dic.values()`                      | Devuelve una vista con los valores.             | `persona.values()`                                         |
| `dic.items()`                       | Devuelve pares `(clave, valor)` en tuplas.      | `persona.items()`                                          |
| `dic.get(clave, valor_por_defecto)` | Evita errores si la clave no existe.            | `persona.get('ciudad', 'N/A')`                             |
| `dic.pop(clave)`                    | Elimina una clave y devuelve su valor.          | `persona.pop('edad')`                                      |
| `dic.popitem()`                     | Elimina y devuelve el √∫ltimo par a√±adido.       | `persona.popitem()`                                        |
| `dic.clear()`                       | Elimina todo el contenido.                      | `persona.clear()`                                          |
| `dic.copy()`                        | Crea una copia superficial del diccionario.     | `copia = persona.copy()`                                   |
| `dic.update(otro_dic)`              | A√±ade o actualiza valores con otro diccionario. | `persona.update({"edad": 27})`                             |

## üß± Crear diccionarios de diferentes formas

```python
# Usando llaves
a = {"x": 1, "y": 2}

# Con dict() y pares clave-valor
b = dict(nombre="Moralens", edad=26)

# Desde listas o tuplas
c = dict([("nombre", "Ana"), ("edad", 22)])
```

## üîÅ Recorrer un diccionario

```python
for clave in persona:
    print(clave, persona[clave])
```

O m√°s completo:

```python
for clave, valor in persona.items():
    print(f"{clave}: {valor}")
```

## üßÆ Ejemplo pr√°ctico

```python
productos = {
    "manzana": 1.2,
    "banana": 0.8,
    "naranja": 1.5
}

# Actualizar valor

# Agregar nuevo

# Recorrer
for fruta, precio in productos.items():
    print(f"{fruta}: ${precio}")
```
